System Control Panel

Простая панель управления компьютером на Python. Открывает браузер, YouTube, проводник, выключает и перезагружает компьютер.

 Установка

 Требования:
- Python 3.6 или новее
- Windows/Linux/macOS

Шаги:
1. Скачай Python** если нет: [python.org](https://python.org)
2. Скачай файлы проекта** (или скопируй код)
3. Запусти программу:
```bash
python 2пр.py
```

 Как пользоваться

 Простая версия (1 файл):
```
2пр.py
```
- Запускаешь файл
- Видишь 6 кнопок
- Нажимаешь что нужно
- Готово!

 Расширенная версия:
```
2прUpdate.py
```
- Есть меню слева (навигация)
- Больше функций (калькулятор, блокнот, 50+ сайтов)
- Настройки (темы, сохранение)
- Поиск по функциям

 Функции

 Основные:
-  Браузер - открывает Google
-  YouTube - открывает Ютуб
-  Проводник - открывает папки
-  Выключить - выключает комп (спросит подтверждение)
-  Перезагрузить - перезапускает комп (спросит)
-  Выход - закрывает программу

 Дополнительные (в расширенной версии):
- Калькулятор, блокнот, терминал
- GitHub, Stack Overflow, Google Drive
- Настройки темной/светлой темы
- Сохранение настроек

 Важно!

 Для выключения/перезагрузки:
1. На Windows: запусти программу "От имени администратора"
   - Правой кнопкой по файлу → "Запуск от имени администратора"
2. На Linux/macOS: запусти через терминал с `sudo`

 Если не работает:
1. Проверь, что Python установлен: `python --version`
2. Если ошибка с правами - запусти от админа
3. Если не открываются сайты - проверь интернет


 Для разработчиков

 Если хочешь добавить свою кнопку (в простой версии):
1. Открой файл `2пр.py`
2. Найди раздел с кнопками
3. Добавь новую кнопку по примеру:

```python
tk.Button(..., text="Моя кнопка", command=my_function, ...)
```

Зависимости (для расширенной версии):
```bash
pip install psutil requests
```

 Поддержка

Проблемы:
1. Проверь права администратора
2. Перезапусти программу
3. Если все равно не работает - напиши, какие ошибки видишь

 Советы

- Используй Ctrl+Q для быстрого выхода
- Ctrl+B открывает браузер
- Ctrl+Y открывает YouTube

- Настрой тему в расширенной версии (Настройки)
